# Unity Design Patterns

A demonstration project showcasing various software design patterns implemented in Unity game development, created for a Game Design course at PUC Minas.

## Description / Descrição

**English:**
Unity Design Patterns is an educational project that demonstrates the implementation of common software design patterns within Unity game development. Created as an exercise minigame for a Game Design course at PUC Minas, this project serves as a practical reference for understanding how design patterns can be applied to solve common game development challenges, improve code organization, and create maintainable game architectures.

**Português:**
Unity Design Patterns é um projeto educacional que demonstra a implementação de padrões de design de software comuns no desenvolvimento de jogos Unity. Criado como um minijogo de exercício para um curso de Game Design na PUC Minas, este projeto serve como referência prática para entender como os padrões de design podem ser aplicados para resolver desafios comuns de desenvolvimento de jogos, melhorar a organização do código e criar arquiteturas de jogos manuteníveis.
